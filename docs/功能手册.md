# 五环共创培训智能体 — 功能手册

## 产品定位

面向教师培训场景的轻量级学习管理系统。帮助班主任组织培训、收集学员产出、凝聚群体共识、追踪训后转化。

## 角色体系

| 角色 | 职责 |
|------|------|
| **管理员** | 配置账号权限、查看全量数据、治理知识库 |
| **班主任** | 运营班级流程、发起共创、生成学员方案、追踪转化 |
| **学员** | 提交学习产出、参与共创、查看个人方案、提交转化成果 |

## 核心功能模块

### 1. 账号登录

- 演示账号一键登录（admin/teacher/student 各一）
- 手动输入账号密码登录
- 角色驱动权限隔离

### 2. 五环学习流程

| 环节 | 教练 | 学员提交内容 | 达标规则 |
|------|------|-------------|---------|
| 深学环 | 深学教练 | 学习批注（看到/理解/迁移） | 点赞≥1/3参训人数 |
| 跟练环 | 跟练教练 | 即时贴（照片+技能说明） | 点赞≥1/3参训人数 |
| 反思环 | 反思教练 | GROW反思案例 | 点赞≥1/3参训人数 |
| 共创环 | 共创教练 | 多行观点（建议+理由+条件） | 收敛投票后生成共识报告 |
| 转化环 | 转化教练 | 转化成果（场景+证据+改进） | 评估合格入库 |

### 3. 共创收敛

- 班主任发起共创主题
- 学员按小组提交观点
- 学员投票（每人每条一票）
- 班主任运行收敛（智能去重、饱和判断）
- 生成《共识报告》，归档当前主题

### 4. 行动计划

- 班主任选择学员生成《个人行动计划》
- 方案包含推荐语、行动步骤、D+30/D+90 追踪节点
- 学员确认方案，按节点提交转化成果
- 转化教练追踪节点到期提醒

### 5. 知识库

- 点赞达标内容自动入库
- 按环节分库查看
- 管理员可编辑、隐藏、删除条目
- 作为总教练生成方案的底层素材

### 6. 模型配置

- 支持接入 OpenAI-compatible 外部模型
- 用于内容审核（质量判断、标签、摘要）
- API Key 本地浏览器存储

## 当前问题

1. **所有 UI 代码集中在 3661 行的单文件组件中**——无法维护
2. **页面内容过量**——仪表盘堆积了大量指标卡、数据看板、运营面板，对班主任和学员造成信息过载
3. **五种彩色标签系统**——视觉杂乱，缺乏统一感
4. **左侧导航栏占用空间**——传统后台风格，不适合轻量应用
5. **大量后台治理功能直接暴露**——如账号管理、权限矩阵、模型配置对学员可见
